1. Static (默认)

Box 1 (Static)

这个盒子遵循正常的文档流。

2. Relative

Box 2 (Relative)

这个绿色的盒子设置了 position: relative; top: 20px; left: 30px;。它相对于自己的原始位置移动了，但它原来的空间依然被保留着，所以这段文字没有往上移动。

3. Absolute (在 Relative 内部)

这是一个父容器，它设置了 position: relative;，成为了一个“锚点”。
Box 3 (Absolute)

橙色的子盒子设置了 position: absolute; bottom: 10px; right: 10px;，所以它被精确定位在父容器的右下角。它已经脱离了文档流。

4. Fixed

Box 4 (Fixed)

紫色的盒子设置了 position: fixed;。它相对于浏览器窗口定位。请尝试滚动页面，你会发现它一直停在右下角。
